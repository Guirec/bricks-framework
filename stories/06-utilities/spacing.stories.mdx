# Spacing

Spacing allow to control margins, padding and gaps. These are based on the spacing settings.

## Margin

|  Class   |    Properties     |
| :------: | :---------------: |
| `m-xss`  | `margin: 0.4rem;` |
|  `m-xs`  | `margin: 0.8rem;` |
|  `m-sm`  | `margin: 1.6rem;` |
|  `m-md`  | `margin: 2.4rem;` |
|  `m-lg`  | `margin: 3.2rem;` |
|  `m-xl`  |  `margin: 4rem;`  |
| `m-2xl`  | `margin: 4.8rem;` |
| `m-3xl`  | `margin: 5.6rem;` |
| `m-4xl`  | `margin: 6.4rem;` |
| `m-5xl`  | `margin: 7.2rem;` |
| `m-6xl`  |  `margin: 8rem;`  |
| `m-auto` |  `margin: auto;`  |
|  `m-0`   |   `margin: 0;`    |

Target a specific side using `margin-{side}-{space}`:

|  Class  |        Properties        |
| :-----: | :----------------------: |
| `mt-xs` |  `margin-top: 0.8rem;`   |
| `mr-sm` | `margin-right: 1.6rem;`  |
| `mb-md` | `margin-bottom: 2.4rem;` |
| `ml-lg` |  `margin-left: 3.2rem;`  |

## Padding

|  Class   |     Properties     |
| :------: | :----------------: |

|  `p-sm`  | `padding: 1.6rem;` |
|  `p-md`  | `padding: 2.4rem;` |

| `p-2xl`  | `padding: 4.8rem;` |

| `p-4xl`  | `padding: 6.4rem;` |

## Gap

|   Class   |   Properties   |
| :-------: | :------------: |
| `gap-xss` | `gap: 0.4rem;` |
| `gap-xs`  | `gap: 0.8rem;` |
| `gap-sm`  | `gap: 1.6rem;` |
| `gap-md`  | `gap: 2.4rem;` |
| `gap-lg`  | `gap: 3.2rem;` |
| `gap-xl`  |  `gap: 4rem;`  |
| `gap-2xl` | `gap: 4.8rem;` |
| `gap-3xl` | `gap: 5.6rem;` |
| `gap-4xl` | `gap: 6.4rem;` |
| `gap-5xl` | `gap: 7.2rem;` |
| `gap-6xl` |  `gap: 8rem;`  |
|  `gap-0`  |   `gap: 0;`    |

Use the `gap-y-{space}` / `gap-x-{space}` classes to modify only the vertical / horizontal gap.
