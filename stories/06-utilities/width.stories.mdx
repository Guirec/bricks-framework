# Width

Utilities for setting the width of an element.

With the default settings, this will be your reference:

|  Class   |     Properties     |
| :------: | :----------------: |
| `w-1/5`  |   `width: 20%;`    |
| `w-1/4`  |   `width: 25%;`    |
| `w-1/3`  | `width: 33.3333%;` |
| `w-1/2`  |   `width: 50%;`    |
| `w-2/3`  | `width: 66.6667%;` |
| `w-3/4`  |   `width: 75%;`    |
| `w-full` |   `width: 100%;`   |
| `w-auto` |   `width: auto;`   |

## Max-width

Max-width values can be used to limit container width. With the default settings, this will be your reference:

|    Class     |      Properties      |
| :----------: | :------------------: |
|  `max-w-xl`  | `max-width: 144rem;` |
| `max-w-full` |  `max-width: 100%;`  |
