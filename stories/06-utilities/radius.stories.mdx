# Radius

Apply a border radius with utility classes can be pretty useful.

With the default settings, this will be your reference:

|          Class          |            Properties            |
| :---------------------: | :------------------------------: |
|       `radius-sm`       |     `border-radius: 0.4rem;`     |
|       `radius-md`       |     `border-radius: 0.8rem;`     |
|      `radius-full`      |      `border-radius: 50em;`      |
|     `radius-circle`     |      `border-radius: 50%;`       |
|       `radius-0`        |       `border-radius: 0;`        |
|   `radius-top-left-0`   |   `border-top-left-radius: 0;`   |
|  `radius-top-right-0`   |  `border-top-right-radius: 0;`   |
| `radius-bottom-left-0`  | `border-bottom-left-radius: 0;`  |
| `radius-bottom-right-0` | `border-bottom-right-radius: 0;` |
