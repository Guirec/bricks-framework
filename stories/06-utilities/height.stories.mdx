# Height

Utilities for setting the height of an element.

With the default settings, this will be your reference:

|  Class   |     Properties      |
| :------: | :-----------------: |
| `h-1/5`  |   `height: 20%;`    |
| `h-1/4`  |   `height: 25%;`    |
| `h-1/3`  | `height: 33.3333%;` |
| `h-1/2`  |   `height: 50%;`    |
| `h-2/3`  | `height: 66.6667%;` |
| `h-3/4`  |   `height: 75%;`    |
| `h-full` |   `height: 100%;`   |
| `h-auto` |   `height: auto;`   |
