# Text

Various typographical aspects can have defaults, utilities, and can be easily customisable.

## Font families

By default there no custom fonts, the base font-family is set to use the system fonts.

|    Name    |      CSS variable       |
| :--------: | :---------------------: |
|   serif    |   `var(--font-serif)`   |
| sans-serif |   `var(--font-sans)`    |
| monospace  | `var(--font-monospace)` |

## Line heights

|  Name  | Value |      CSS variable       |
| :----: | :---: | :---------------------: |
|  none  |   1   |  `var(--leading-none)`  |
| tight  | 1.25  | `var(--leading-tight)`  |
|  snug  | 1.375 |  `var(--leading-snug)`  |
| normal |  1.5  | `var(--leading-normal)` |
