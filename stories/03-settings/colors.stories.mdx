# Colors

All colors are css variables that can be used as such in (s)css files (`--color-{color-value}`) and there are also utility classes generated with the pattern `text-{color-value}` for text color or `bg-{color-value}` for background color.

## Main colors

They can be defined as the "brand" colors.
