# Spacing

Spacing in a page is a very critical point. Being consistent is a very great key to a good looking interface, so set the rules with a range of spacing options lead to the greatest level of consistancy.

A multiple of 8 to increment each spacing values is the default. But 8px can be a little too much as minimum, so we start with 4px for the smallest space value.

## Map

| Name | Value  |    CSS variable    |
| :--: | :----: | :----------------: |
|  0   |   0    |  `var(--space-0)`  |
| xxs  | 0.4rem | `var(--space-xxs)` |
|  xs  | 0.8rem | `var(--space-xs)`  |
|  sm  | 1.6rem | `var(--space-sm)`  |
|  md  | 2.4rem | `var(--space-md)`  |
|  lg  | 3.2rem | `var(--space-lg)`  |
|  xl  |  4rem  | `var(--space-xl)`  |
| 2xl  | 4.8rem | `var(--space-2xl)` |
| 3xl  | 5.6rem | `var(--space-3xl)` |
| 4xl  | 6.4rem | `var(--space-4xl)` |
| 5xl  | 7.2rem | `var(--space-5xl)` |
| 6xl  |  8rem  | `var(--space-6xl)` |

## Usage

Spacing values can be used to define margin, padding and gaps.
