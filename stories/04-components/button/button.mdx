# Button

Button class can be applied on `<button>`, `<a>` or `<input>` html tags. Several variants and state are available.

If there is a `disabled` attribute, the style adjust accordingly, for all the styles and states.

## Styles

|  Styles  |        Class        |
| :------: | :-----------------: |
| Default  |        `btn`        |
| Reversed | `btn btn--reversed` |
|   link   |   `btn btn--link`   |

## State

All the button styles can have a loading animation added, for that you have to add the corresponding variant class to your exiting ones.

|  State  | Class variant |
| :-----: | :-----------: |
| Loading |  `--loading`  |
